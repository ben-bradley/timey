# Timey

Timey makes converting human-readable times to milliseconds easy and clean.

## Installation

`npm install timey`

-or-

`npm install ben-bradley/timey`

## Use

```
var ms = require('timey');

console.log(ms(10).years);  // => 315360000000
console.log(ms(10).y);      // => 315360000000
```
